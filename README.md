# trpp-lab5-ci

Простой калькулятор (HTML + JS) с Jest-тестами и CI/CD на GitHub Actions.

## Локально

```bash
npm install
npm test
npm run build
```

Откройте `index.html` в браузере после `npm run build`.

## CI

Push в `main` → тесты → деплой на GitHub Pages.

Демо: https://moroz-js.github.io/trpp-lab5-ci/
